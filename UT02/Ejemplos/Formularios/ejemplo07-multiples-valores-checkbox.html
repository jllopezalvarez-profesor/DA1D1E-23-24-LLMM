Ejemplo de recogida de múltiples valores - Checkbox

Este formulario tiene varios input type="checkbox" para seleccionar colores.

El usuario puede elegir varios colores, no sólo uno.

Se enviará al servidor múltiples valores para la misma variable "color".

Aunque tengan el mismo atributo "name" se pueden marcar de forma independiente.

Sólo los iput type="radio" se excluyen entre ellos si tienen el mismo nombre.

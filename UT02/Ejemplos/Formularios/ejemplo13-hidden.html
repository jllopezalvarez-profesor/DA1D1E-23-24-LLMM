Ejemplo de formulario con campos ocultos

En el siguiente formulario, que simula un formulario para editar los datos personales de una persona, además hay un campo oculto donde está el id de esa persona en una hipotética base de datos.

Este tipo de formularios se genera dinámicamente en el servidor, y vienen con valores precargados en los campos, para que el usuario los modifique.

Al enviar el formulario, junto a los campos visibles se envía el ID, para que el servidor pueda asociar el nombre, apellidos y fecha de nacimiento con una persona de la base de datos.

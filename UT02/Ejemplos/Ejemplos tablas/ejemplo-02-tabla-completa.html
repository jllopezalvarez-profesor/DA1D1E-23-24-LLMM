Ejemplo de tabla HTML completa

Esta tabla utiliza la estructura más completa de tabla. Se utilizan:

  • Elemento table, que delimita la tabla
  • Elemento caption, para el título de tabla
  • Elemento thead, que delimita la cabecera
  • Elemento tbody, que delimita el cuerpo de la tabla
  • Elemento tfoot, que delimita el pié de la tabla
  • Elemento tr, que delimita cada fila
  • Elemento th, que delimita cada celda de cabecera
  • Elemento td, que delimita cada celda de datos

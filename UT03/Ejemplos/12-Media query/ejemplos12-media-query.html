Grid con tamaños absolutos (px)

Sólo display:grid

Esto no tiene un efecto sobre el contenido. Aparentemente, los elementos interiores no se ven alterados. Para que se dispongan en forma de rejilla hay que definir filas y columnas.

Los contenidos se ven centrados vertical y horizontalmente porque cada elemento del grid es a su vez un elemento flex con el contenido justificado en eje primario al centro, y con alineado en eje secundario al centro

display: grid;

Filas automáticas (indefinidas / infinitas)

Cuadrícula de tres columnas iguales, y usamos grid-auto-rows en las filas.

La propiedad grid-auto-rows permite definir el alto de las filas que no se han especifocado con grid-template-rows.

En este caso, la primera y segunda fila se han definido con template, y el resto de filas usan el valor definido en grid-auto-rows.

El uso de unidades fr en grid-auto-rows no es trivial. Necesita que se defina al menos una fila con tamaño fr en grid-template-rows para poder tomarla como referencia. 

display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: 100px 300px;
            grid-auto-rows: 200px;

Minmax, auto-fill, auto-fit

Minmax.

La función minmax permite especificar un máximo y un mínimo para una fila o columna

En este caso, la segunda columna tendrá un tamño de 500px, siempre que quepa en la página. Si no cabe, se reducirá hasta un mínimo de 300px, nunca será menor.

display: grid;
            grid-template-columns: 300px minmax(300px, 500px) 1fr;

Espaciado / gutters

row-gap y column-gap

Define el espacio de separación entre las celdas del grid, horizontal / verticalmente respectivamente.

Ojo, el espaciado es interior, no exterior, por eso está pegado el grid al elemento anterior, que contiene el código CSS

display: grid;
            grid-template-columns: repeat(5, 1fr);
            grid-template-rows: 1fr;
            grid-auto-rows: 1fr;
            row-gap: 10px;
            column-gap: 50px;

Ubicación de elementos

En todos los ejemplos anteriores cada item ocupa un elemento en la rejilla, pero grid permite ubicar los items en posiciones concretas, que ocupen más de una celda, o incluso solapar celdas.

Es importante entender que, cuando indicamos inicio y fin de un elemento en grid, no nos referimos a los huecos o celdas, sino a las líneas.

Un grid de 5 columnas y 4 filas tendrá 6 líneas de grid verticales (de la 1 a la 6) y 5 horizontales (de la 1 a la 5)

Todos los ejemplos de esta sección serán con un grid con la siguiente estructura:

display: grid;
            grid-template-columns: repeat(5, 1fr);

Y trabajaremos con los elementos

Elemento que ocupa varias celdas en horizontal

Se usan las propiedades grid-column-start y grid-column-end

Ojo, estas propiedades referencian números de línea, no celdas.

/* Item 1 */
            grid-column-start: 1;
            grid-column-end: 3;
            /* Item 7 */
            grid-column-start: 3;
            grid-column-end: 6;

Usando valores de línea negativos

Para estas propiedades, podemos usar números negativos.

En la rejilla, las líneas se numeran de 1 hasta (nº de columnas/filas)+1 en el sentido de lectura, y de -1 a -((nº de columnas/filas)+1) desde el final en sentido contrario.

/* item 1*/
            grid-column-start: 3;
            grid-column-end: -2;
            /* item 8 */
            grid-column-start: -1;
            grid-column-end: -5;

Grid por áreas

Se usa "grid-template-areas" para definir las áreas y "grid-area" para indicar en qué área de las definidas se ubica un contenido

Hay que definir también, al menos, el ancho de las columnas

Estructura con cabecera, menú izquierdo, contenido, y pie

Se define la estructura en el contenedor grid:

display: grid;
            grid-template-areas:
            "cabecera cabecera"
            "menu principal"
            "pie pie";
            grid-template-columns: 1fr 3fr;
            grid-template-rows: 200px 600px 100px;

Y se asigna cada elemento al área correspondiente


            /* Cabecera */
            grid-area: cabecera;
            background-color: rgb(247, 140, 140);
            /* Menu */
            grid-area: menu;
            background-color: rgb(195, 255, 255);
            /* Cuerpo ppal */
            grid-area: principal;
            background-color: lightgreen;
            /* Pie */
            grid-area: pie;
            background-color: rgb(221, 171, 255);

Ejemplos background-image

Comportamiento por defecto y repetición del fondo

Comportamiento por defecto

background-image: url("fondo1.jpg");

Por defecto el fondo se repite, para rellenar el área. Es el equivalente a usar "repeat" para "background-repeat". En fondos diseñados para que se repitan, puede darse la impresión de que se tiene una imagen grande de fondo, cuando en realidad es una pequeña que se repite.

Sin repetición

background-image: url("fondo1.jpg");
            background-repeat: no-repeat;
            background-color: aqua;

Si no se repite, se mostrará el color de fondo en las zonas no cubiertas por la imagen.

Repetición horizontal (eje x)

background-image: url("fondo1.jpg");
                        background-repeat: repeat-x;
                        background-color: aqua;

Repetición en el eje x.

Repetición vertical (eje y)

background-image: url("fondo1.jpg");
                                    background-repeat: repeat-y;
                                    background-color: aqua;

Repetición en el eje y.

Repetición redondeando tamaño

background-image: url("fondo1.jpg");
                                                background-repeat: round;
                                                background-color: aqua;

Igual que repeat, pero "estira" la imagen para que no se "corte" en los extremos del elemento.

Repetición con espaciado

background-image: url("fondo1.jpg");
                                                            background-repeat: space;
                                                            background-color: aqua;

Igual que repeat, pero introduce espacios entre imágenes para que no se "corte" en los extremos del elemento.

Posicionamiento de imagen

Posicionamiento con unidades

background-image: url("fondo1.jpg");
background-repeat: no-repeat;
            background-position: 100px, 100px;
            background-color: aqua;

El fondo no se repite y lo posicionamos a 100px del borde superior y del lado izquierdo. Puede usarse porcentajes. En ese caso el porcentaje se refiere a las dimensiones del elemento al que se aplica el fondo.

Posicionamiento con palabras clave

url("fondo1.jpg");
            background-repeat: no-repeat;
            background-position: bottom right;
            background-color: aqua;

Se pueden usar palabras clave (top, left, right, bottom y center) para indicar dónde se debe colocar la imagen.

Imagen como fondo de página

Comportamiento por defecto - scroll

background-image: url("fondo2.jpg");
            background-attachment: scroll;

Por defecto el fondo "sigue" el scroll de la página, se mueve al desplazarse por ella.

Ejemplo de página con fondo "scroll".

Fondo fijo

background-image: url("fondo2.jpg");
                        background-attachment: fixed;

Podemos hacer que el contenido se "deslice" sobre el fondo, que se mantiene fijo.

Ejemplo de página con fondo "fixed".

Estirando el fondo - background-size

Usando tamaño fijo

background-image: url("fondo1.jpg");
            background-size: 70px 150px;

Se puede cambiar el tamaño indicando dimensiones absolutas. Si usamos sólo una dimensión se ajusta el ancho, y se mantienen proporciones. Si usamos dos, la imagen se puede deformar.

Usando porcentaje

background-image: url("fondo1.jpg");
            background-size: 50%;

Si usamos un porcentaje, el porcentaje se refiere al ancho / alto del elemento al que aplicamos el fondo. Se pueden usar también uno o dos valores, y ocurre igual que con fijas, se puede deformar la imagen.

Ajustar al ancho

background-image: url("fondo1.jpg");
            background-size: cover;

La imagen de fondo se estira para que ocupe siempre el ancho del elemento.

Ajustar al alto

background-image: url("fondo1.jpg");
            background-size: contain;

La imagen de fondo se estira para que ocupe siempre el alto del elemento.

Degradados

Se pueden usar degradados en lugar de imágenes. Ver la página de ejemplos de degradados para más informacion.

Ejemplo de fondo degradado

background-image: linear-gradient(90deg, red, blue);
